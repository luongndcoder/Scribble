Hướng dẫn cài đặt

Hướng dẫn cài đặt đầy đủ

Làm theo các bước sau để Scribble hoạt động trong vòng 5 phút. Ảnh chụp màn hình kèm theo mỗi bước.

Phase 1

Tải & Cài đặt

Cài Scribble lên máy của bạn

1

Tải file DMG

Nhấn nút Tải xuống ở phần bên trên để tải file .dmg. Tìm file trong thư mục Downloads của bạn.

2

Xử lý cảnh báo "Not Opened"

Double-click file .dmg. macOS có thể hiện cảnh báo "Scribble_1.0.0_aarch64.dmg Not Opened" vì file chưa được ký. Nhấn "Done" — chúng ta sẽ sửa ở bước tiếp.

3

Cho phép trong Privacy & Security

Mở System Settings → Privacy & Security. Cuộn xuống tìm thông báo Scribble và nhấn "Open Anyway" để cho phép DMG chạy.

Hoặc chạy lệnh: xattr -cr ~/Downloads/Scribble*.dmg trong Terminal
4

Xác nhận mở DMG

Hộp thoại xác nhận sẽ xuất hiện. Nhấn "Open Anyway" và nhập mật khẩu để tiếp tục.

5

Kéo Scribble vào Applications

DMG sẽ mở hiện icon Scribble và shortcut thư mục Applications. Kéo icon Scribble vào Applications để cài đặt.

6

Xử lý cảnh báo "Not Opened" của ứng dụng

Khi bạn lần đầu mở Scribble từ Applications, macOS hiện thêm cảnh báo "Scribble Not Opened". Nhấn "Done" — cần cho phép qua Privacy & Security lần nữa.

7

Cho phép ứng dụng trong Privacy & Security

Quay lại System Settings → Privacy & Security. Bạn sẽ thấy Scribble bị chặn. Nhấn "Open Anyway" một lần nữa.

Hoặc chạy: xattr -cr /Applications/Scribble.app để bỏ qua bước này
8

Xác nhận & mở ứng dụng

Hộp thoại Gatekeeper xuất hiện lần cuối. Nhấn "Open Anyway" và xác thực bằng mật khẩu hoặc Touch ID.

✓

Đã cài xong Scribble!

Ứng dụng khởi chạy thành công. Bạn sẽ thấy giao diện chính Scribble. Tiếp tục Phase 2 để thiết lập API key.

1

Tải bộ cài đặt

Nhấn nút Tải xuống ở phần bên trên để tải file .exe. Trình duyệt có thể hiện cảnh báo file ít được tải — điều này bình thường với ứng dụng mới.

2

Nhấn menu "⋯" → Keep

Nhấn menu ba chấm (⋯) trên thanh cảnh báo tải xuống. Chọn "Keep" từ dropdown để giữ file thay vì xóa.

3

Nhấn "Keep anyway"

Xác nhận lần hai xuất hiện. Nhấn "Keep anyway" để xác nhận lưu bộ cài.

4

Mở bộ cài đặt

Vào thư mục Downloads và double-click Scribble_1.0.0_x64-setup.exe để bắt đầu cài đặt.

5

Bỏ qua SmartScreen — nhấn "Show more"

Windows Defender SmartScreen sẽ hiện "This app might harm your device". Nhấn "Show more ∨" để hiện tùy chọn tiếp tục.

6

Nhấn "Install anyway"

Sau khi nhấn "Show more", nút "Install anyway" xuất hiện. Nhấn để tiếp tục cài đặt.

✓

Cài đặt hoàn tất!

Trình cài đặt sẽ hướng dẫn bạn qua các bước còn lại. Chọn ngôn ngữ, vị trí cài đặt, và Scribble sẽ sẵn sàng trong vài giây. Tiếp tục Phase 2 để thiết lập API key.

Phase 2

Lấy API Key NVIDIA miễn phí

Cần thiết cho tính năng phiên âm và AI (có gói miễn phí)

1

Truy cập build.nvidia.com

Vào build.nvidia.com và nhấn nút "Login" ở góc trên bên phải. Nếu chưa có tài khoản, bạn cần tạo mới.

2

Tạo tài khoản

Điền email, mật khẩu, hoàn thành hCaptcha, rồi nhấn "Create Account". Bạn cũng có thể đăng ký qua Google hoặc GitHub.

3

Xác minh email

NVIDIA sẽ gửi mã 6 chữ số đến email. Nhập mã và nhấn "Continue".

4

Hoàn tất đăng ký

Xem lại cài đặt tùy chọn (không bắt buộc). Nhấn "Submit" để hoàn tất tài khoản. Bạn sẽ được chuyển đến trang quản trị NVIDIA.

5

Tạo NVIDIA Cloud Account

Bạn sẽ được yêu cầu tạo NVIDIA Cloud Account. Nhập Cloud Account Name (ví dụ: tên bạn hoặc tổ chức), rồi nhấn "Create NVIDIA Cloud Account".

6

Nhấn "Verify" để được cấp quyền API

Sau khi đăng nhập, bạn sẽ thấy banner vàng phía trên: "Please verify your account to get API access". Nhấn nút "Verify" bên phải banner.

Bước này bắt buộc — bạn không thể tạo API key mà không xác minh số điện thoại.
7

Xác minh số điện thoại

Hộp thoại yêu cầu số điện thoại sẽ xuất hiện. Chọn Location, nhập Phone Number, rồi nhấn "Send Code via SMS". Nhập mã OTP nhận được để hoàn tất xác minh.

8

Đi tới API Keys

Nhấn icon hồ sơ ở góc trên bên phải, rồi chọn "API Keys" từ menu dropdown.

9

Tạo API Key

Nhấn "Generate API Key". Sao chép key ngay lập tức và lưu ở nơi an toàn — bạn sẽ không thể xem lại nó!

Gói miễn phí cho bạn 1.000 API credit/tháng — đủ dùng cho cá nhân.
Phase 3

Cấu hình Scribble

Dán API key và thiết lập mô hình AI

1

Mở Settings → Riva (Phiên âm)

Mở Scribble, nhấn biểu tượng ⚙️ Settings ở góc trên bên phải. Trong phần Riva, bạn sẽ thấy các trường cấu hình kết nối.

2

Nhập Riva API Key & Server URL

Dán NVIDIA API key vào trường API Key. Sau đó nhập server URL và nhấn "Test Connection" để kiểm tra.

API Key nvapi-xxxx (your NVIDIA API key)
Server URL grpc.nvcf.nvidia.com:443
3

Cấu hình AI Assistant API Key

Cuộn xuống phần AI Assistant. Dán cùng NVIDIA API key vào trường API Key.

4

Đặt AI Server URL & Model

Nhập địa chỉ server và tên model trong phần AI Assistant. Nhấn "Save" để áp dụng tất cả cài đặt.

Server URL https://integrate.api.nvidia.com/v1
Model deepseek-ai/deepseek-v3.2
Bạn có thể dùng bất kỳ AI model tương thích nào. DeepSeek v3.2 được khuyên dùng cho kết quả tốt nhất.
Phase 4

Bắt đầu ghi âm

Cấp quyền và bắt đầu phiên đầu tiên

1

Cấp quyền Screen & System Audio macOS only

Khi bạn nhấn "Record" lần đầu, macOS sẽ yêu cầu quyền Screen & System Audio Recording. Nhấn "Open System Settings" để đến thẳng bảng cấp quyền.

2

Bật Scribble trong System Settings macOS only

Trong System Settings → Privacy & Security → Screen & System Audio Recording, tìm Scribble trong danh sách và bật ON. Có thể cần nhập mật khẩu.

✓

Hoàn tất! Bắt đầu cuộc họp đầu tiên

Tạo cuộc họp mới, nhấn Record, và xem phiên âm thời gian thực với nhận diện người nói xuất hiện ngay lập tức. Scribble tự động nhận diện người nói và phát trực tiếp bản ghi.

Câu hỏi thường gặp

Scribble có an toàn không?

Có. Scribble 100% mã nguồn mở — bạn có thể kiểm tra từng dòng code trên GitHub. Cảnh báo bảo mật xuất hiện vì ứng dụng chưa được ký bằng chứng chỉ thương mại.

Tại sao phần mềm diệt virus báo Scribble?

Scribble dùng backend Python (đóng gói bằng PyInstaller) mà một số phần mềm diệt virus nhầm báo là mã độc. Đây là lỗi dương tính giả phổ biến ảnh hưởng nhiều ứng dụng Python.

Có cần trả phí cho API không?

Không. Cả NVIDIA và Soniox đều cung cấp gói miễn phí. NVIDIA tại build.nvidia.com, Soniox tại soniox.com. Đăng ký và tạo API key miễn phí.
