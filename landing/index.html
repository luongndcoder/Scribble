Features

Everything you need for
perfect meetings

No cloud dependency. No subscription. Just powerful AI running on your machine.

Real-time Transcription

Nvidia Riva streaming gRPC with sub-second latency. Words appear the moment they're spoken.

Speaker Identification

ONNX Runtime + CAM++ diarization identifies who's speaking. Each voice auto-labeled in real-time.

Live Translation

Real-time cabin translation into 12+ languages. Translations stream below each transcript line.

AI Meeting Minutes

One-click summaries: title, key points, action items, decisions, risks. Export as Markdown or Word.

Auto-save & Recovery

Transcripts save incrementally to SQLite. Even if the app crashes, your meeting data stays safe.

Privacy First

Everything runs locally. API keys stored on your machine only. No data leaves without your consent.

Setup Guide

Complete setup walkthrough

Follow these steps to get Scribble running in under 5 minutes. Screenshots included for every step.

Phase 1

Download & Install

Get Scribble on your machine

1

Download the DMG file

Go to GitHub Releases and download the .dmg file (e.g. Scribble_1.1.0_aarch64.dmg). Find it in your Downloads folder.

2

Handle "Not Opened" warning

Double-click the .dmg file. macOS may show a "Scribble_1.1.0_aarch64.dmg Not Opened" warning because it's unsigned. Click "Done" for now — we'll fix this in the next step.

3

Allow in Privacy & Security

Open System Settings → Privacy & Security. Scroll down to find the Scribble message and click "Open Anyway" to allow the DMG to run.

Alternative: run xattr -cr ~/Downloads/Scribble*.dmg in Terminal
4

Confirm opening the DMG

A confirmation dialog will appear asking to open the DMG. Click "Open Anyway" and enter your password to proceed.

5

Drag Scribble to Applications

The DMG will open showing the Scribble app icon and an Applications folder shortcut. Drag the Scribble icon into Applications to install.

6

Handle app "Not Opened" warning

When you first launch Scribble from Applications, macOS shows another "Scribble Not Opened" warning. Click "Done" — we need to allow it through Privacy & Security again.

7

Allow app in Privacy & Security

Go back to System Settings → Privacy & Security. You'll see the Scribble app has been blocked. Click "Open Anyway" once more.

Or run: xattr -cr /Applications/Scribble.app to skip this step
8

Confirm & open the app

The Gatekeeper dialog appears one last time. Click "Open Anyway" and authenticate with your password or Touch ID.

✓

Scribble is installed!

The app launches successfully. You'll see the main Scribble interface. Continue to Phase 2 to set up your API key.

1

Download the installer

Go to GitHub Releases and download Scribble_1.1.0_x64-setup.exe. Your browser will show a warning that it's not commonly downloaded — this is normal for new apps.

2

Click the "⋯" menu → Keep

Click the three-dot menu (⋯) on the download warning bar. Select "Keep" from the dropdown to keep the file instead of deleting it.

3

Click "Keep anyway"

A second confirmation appears. Click "Keep anyway" to confirm saving the installer.

4

Open the installer

Go to your Downloads folder and double-click Scribble_1.1.0_x64-setup.exe to start the installation.

5

Bypass SmartScreen — click "Show more"

Windows Defender SmartScreen will show "This app might harm your device". Click "Show more ∨" to reveal the option to proceed.

6

Click "Install anyway"

After clicking "Show more", the "Install anyway" button appears. Click it to proceed with the installation.

✓

Installation complete!

The installer wizard will guide you through the remaining steps. Choose your language, install location, and Scribble will be ready in seconds. Continue to Phase 2 to set up your API key.

Phase 2

Get your free NVIDIA API Key

Required for speech-to-text and AI features (free tier available)

1

Go to build.nvidia.com

Visit build.nvidia.com and click the "Login" button in the top-right corner. If you don't have an account yet, you'll need to create one.

2

Create your account

Fill in your email, password, complete the hCaptcha, then click "Create Account". You can also sign up with Google or GitHub.

3

Verify your email

NVIDIA will send a 6-digit verification code to your email. Enter the code and click "Continue".

4

Complete registration

Review your preference settings (optional). Click "Submit" to finalize your account. You'll be redirected to the NVIDIA dashboard.

5

Create NVIDIA Cloud Account

You'll be asked to create an NVIDIA Cloud Account. Enter a Cloud Account Name (e.g. your name or organization), then click "Create NVIDIA Cloud Account".

6

Click "Verify" to get API access

After logging in, you'll see a yellow banner at the top: "Please verify your account to get API access". Click the "Verify" button on the right side of the banner.

This step is required — you cannot generate API keys without verifying your phone number.
7

Verify your phone number

A dialog will appear asking for your phone number. Select your Location, enter your Phone Number, then click "Send Code via SMS". Enter the OTP code you receive to complete verification.

8

Navigate to API Keys

Click your profile icon in the top-right corner, then select "API Keys" from the dropdown menu.

9

Generate your API Key

Click "Generate API Key". Copy the key immediately and save it somewhere safe — you won't be able to see it again!

The free tier gives you 1,000 API credits/month — plenty for personal use.
Phase 3

Configure Scribble

Paste your API key and set up the AI model

1

Open Settings → Riva (Speech-to-Text)

Launch Scribble, click the ⚙️ Settings icon in the top-right corner. In the Riva section, you'll see the connection configuration fields.

2

Enter Riva API Key & Server URL

Paste your NVIDIA API key into the API Key field. Then enter the server URL and click "Test Connection" to verify.

API Key nvapi-xxxx (your NVIDIA API key)
Server URL grpc.nvcf.nvidia.com:443
3

Configure AI Assistant API Key

Scroll down to the AI Assistant section. Paste the same NVIDIA API key into the API Key field.

4

Set AI Server URL & Model

Enter the server address and model name in the AI Assistant section. Click "Save" to apply all settings.

Server URL https://integrate.api.nvidia.com/v1
Model deepseek-ai/deepseek-v3.2
You can use any compatible AI model. DeepSeek v3.2 is recommended for best results.
Phase 4

Start Recording

Grant permissions and begin your first session

1

Grant Screen & System Audio macOS only

When you first click "Record", macOS will ask for Screen & System Audio Recording permission. Click "Open System Settings" to go directly to the permission panel.

2

Enable Scribble in System Settings macOS only

In System Settings → Privacy & Security → Screen & System Audio Recording, find Scribble in the list and toggle it ON. You may need to enter your password.

✓

You're all set! Start your first meeting

Create a new meeting, hit Record, and watch real-time transcription with speaker diarization appear instantly. Scribble identifies speakers automatically and streams the transcript live.

Frequently Asked Questions

Is Scribble safe to use?

Yes. Scribble is 100% open source — you can inspect every line of code on GitHub. Security warnings appear because the app has not been signed with a commercial code signing certificate.

Why does my antivirus flag Scribble?

Scribble uses a Python backend (packaged with PyInstaller) that some antivirus engines mistakenly flag. This is a well-known false positive affecting many Python applications.

Do I need to pay for the NVIDIA API?

No. NVIDIA offers a free tier at build.nvidia.com that provides enough credits for personal use. Sign up and generate an API key for free.
